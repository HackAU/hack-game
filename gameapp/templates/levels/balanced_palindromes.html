Balanced Palindromes

We define a balanced sequence of parenthesis as follows:

- Empty string is a balanced sequence of parenthesis.

- If a and b are perfectly balanced parenthesis, so is (a)b.


A palindrome is such a string that it's equal to its reversed. Note that empty string is a palindrome.

Calculate the number of strings of length up to 16 (inclusive), that are both a balanced sequence of parenthesis and a palindrome.
